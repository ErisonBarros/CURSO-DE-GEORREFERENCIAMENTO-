# CURSO DE GEORREFERENCIAMENTO

Bem-vindo ao **Curso de Georreferenciamento**!

Este repositório reúne os materiais, exercícios e conteúdos desenvolvidos para o curso, com o objetivo de fornecer uma base sólida sobre conceitos, técnicas e aplicações do georreferenciamento em diversas áreas.

## Objetivo do Curso

O curso tem como finalidade apresentar aos participantes os fundamentos teóricos e práticos do georreferenciamento, abordando as principais ferramentas, legislações e metodologias utilizadas no Brasil.

## Conteúdo Programático

- Introdução ao Georreferenciamento
- Sistemas de Coordenadas Geográficas
- Fundamentos de Geodésia
- Normas e Legislações aplicáveis
- Ferramentas e Softwares Utilizados
- Coleta de Dados em Campo
- Processamento de Dados
- Geração de Plantas e Relatórios Técnicos
- Estudos de Casos e Aplicações Práticas

## Público-Alvo

Este curso é direcionado a estudantes, profissionais e interessados nas áreas de agrimensura, topografia, engenharia civil, cartografia, geografia e áreas correlatas.

## Pré-requisitos

- Conhecimentos básicos de informática
- Noções de cartografia e geografia (desejável, mas não obrigatório)

## Como Utilizar este Repositório

Neste repositório você encontrará:
- Material didático (apostilas, slides e artigos)
- Exercícios práticos e exemplos de aplicação
- Scripts e arquivos de apoio
- Links para softwares e ferramentas úteis
